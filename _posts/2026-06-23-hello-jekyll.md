### Jekyll 是什么？

Jekyll 是一个静态网站生成器，可以把 Markdown 文件转换成完整的静态网站。

### 为什么用 Jekyll？

- **简单易用**：用 Markdown 写文章，不用管数据库
- **免费托管**：GitHub Pages 原生支持
- **版本控制**：所有内容都在 Git 里
- **加载速度快**：纯静态页面，没有后端
